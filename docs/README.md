## ğŸ¨ Design Philosophy

The website features an **oil painting aesthetic** with:
- Warm, earthy color palette (browns, golds, beiges)
- Textured background mimicking canvas
- Elegant serif typography (Georgia, Times New Roman)
- Smooth transitions and hover effects
- Responsive design for all screen sizes

This artistic style reflects the interpretability theme - making the "invisible" visible, much like an artist revealing hidden beauty.
